A landing page conta com:

Navbar fixa com efeito de vidro (glassmorphism)
Seção Hero com mockup de celular animado
Cards de funcionalidades em grid responsivo
Seção CTA com botões de download
Animações CSS suaves (fade-up e float)
Design 100% responsivo para mobile e desktop

<h1>Conceitos Praticados</h1>
Este projeto foi desenvolvido com foco educacional, cobrindo os seguintes conceitos:

Variáveis CSS com :root para gerenciar cores globalmente
Flexbox para alinhar elementos lado a lado
CSS Grid com auto-fit e minmax() para layout responsivo automático
Animações com @keyframes, animation-delay e transition
Pseudo-elementos e efeitos de hover
Gradientes em texto com -webkit-background-clip
Box-shadow em múltiplas camadas para profundidade
Media queries para adaptação em dispositivos móveis
Semântica HTML5 com <nav>, <section>, <footer>


📱 Responsividade
A página se adapta automaticamente para diferentes tamanhos de tela:

✅ Desktop (1280px+)
✅ Tablet (768px – 1279px)
✅ Mobile (até 767px)
